## 任何一门技术的学习都不是一日之功！脚踏实地一点点的积累练习比什么都强。

## 如果要把.gitignore 里忽略的文件强制添加上去，可以使用  
git add -f .project  
git gitconfig -l 列出所有配置项  
## 查看提交历史  
git log  
## -p 选项展开显示每次提交的内容差异，用 -2 则仅显示最近的两次更新  
git log -p -2
